Install CLI

Requirements

  • Windows 10, Windows 8, Mac OSX, Linux
  • Node.js v10 or higher
  • Minumum RAM required: 2 GB

Installation(BASIC)

For Mac

$ brew install bolt.CLI

For Windows(NPM)

$ npm install bolt.CLI

For Linux

$ yarn install bolt.CLI
